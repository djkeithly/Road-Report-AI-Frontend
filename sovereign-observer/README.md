# Sovereign Observer

This is the current frontend for the Road Report AI project.

Stack:

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- Motion
- Lucide React
- Recharts

## Local Demo

```bash
npm install
npm run dev
```

The dev server runs on:

```text
http://localhost:3000
```

## Notes

- This app is currently the primary UI/demo experience for the repo.
- The older Vue implementation remains in `../road-report-site` as a legacy reference for backend-connected behavior and prior functionality.
- `.env` setup is not currently required just to demo the UI in its present state.
