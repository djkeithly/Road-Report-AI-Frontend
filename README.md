This repo currently contains two frontend implementations:

- [sovereign-observer](C:\Users\kadeo\Downloads\Road-Report-AI-Frontend-main\sovereign-observer) is the current frontend and the one to demo first.
- [road-report-site](C:\Users\kadeo\Downloads\Road-Report-AI-Frontend-main\road-report-site) is the legacy Vue frontend kept for backend integration reference and older connected functionality.

Project notes and weekly progress remain in [Weekly Progress](C:\Users\kadeo\Downloads\Road-Report-AI-Frontend-main\Weekly Progress).

## Current Frontend

The active UI is the React + TypeScript Vite app in [sovereign-observer](C:\Users\kadeo\Downloads\Road-Report-AI-Frontend-main\sovereign-observer).

Run it locally:

```bash
cd sovereign-observer
npm install
npm run dev
```

Default local URL:

```text
http://localhost:3000
```
